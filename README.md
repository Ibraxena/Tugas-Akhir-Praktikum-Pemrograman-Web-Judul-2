Halo! Ini adalah proyek website portofolio pribadi yang saya buat untuk menampilkan profil diri, informasi akademik, serta minat dan keahlian saya di bidang Artificial Intelligence, Game Development, dan Data Analysis.
Website ini dikembangkan menggunakan HTML dan CSS sebagai bagian dari latihan pengembangan front-end sekaligus media representasi profesional diri saya secara digital.

Website ini saya rancang untuk memperkenalkan siapa saya, latar belakang pendidikan saya di Teknik Informatika Universitas Lampung, serta ketertarikan saya dalam dunia AI dan pengembangan game.
Halaman ini memiliki beberapa bagian utama:
  - Header yang menampilkan nama dan deskripsi singkat saya.
  - Tentang Saya, berisi ringkasan latar belakang dan minat akademik saya.
  - Informasi Pribadi yang mencantumkan data penting seperti nama, tanggal lahir, lokasi, dan alamat email.
  - Keahlian & Minat yang disusun dalam bentuk kartu berisi bidang yang paling saya tekuni.
  - Footer sederhana yang menjadi penutup halaman dengan sentuhan personal.

Struktur proyeknya cukup sederhana, terdiri dari file:
  - index.html sebagai struktur utama website.
  - style.css sebagai file pengatur tampilan dan gaya visual.
  - Foto Diri Wisuda.jpg sebagai foto profil utama yang digunakan di halaman.

Dalam pembuatan website ini, saya menggunakan:
  - HTML5 untuk menyusun struktur konten.
  - CSS3 untuk mengatur gaya visual dan tampilan responsif.
  - Flexbox untuk mengatur layout yang adaptif di berbagai ukuran layar.
  - Linear Gradient pada bagian header agar tampak lebih menarik.
  - Efek Hover dan Shadow untuk memberikan kesan interaktif pada elemen tertentu.
